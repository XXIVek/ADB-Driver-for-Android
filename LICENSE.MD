## 2. Предмет соглашения

2.1. Программное обеспечение представляет собой внешнюю компоненту (DLL-библиотеку) для 1С:Предприятие 8.3, обеспечивающую обмен файлами между компьютером на базе Windows и Android-устройством через интерфейс USB/ADB (Android Debug Bridge).

2.2. Состав Программного обеспечения:
- Динамическая библиотека `ADBFileDriver.dll`;
- Вспомогательные утилиты и файлы конфигурации;
- Документация по установке и использованию.
