# ADBFileDriver — Внешняя компонента для 1С:Предприятие 8.3

## Обмен файлами через USB/ADB

Внешняя компонента `ADBFileDriver` предназначена для обмена файлами между компьютером под управлением Windows и устройством на базе Android через USB-кабель с использованием Android Debug Bridge (ADB).

---

## Возможности

- Подключение устройства Android через USB
- Загрузка файлов с ПК на телефон (в папку `/Download/`)
- Скачивание файлов с телефона на ПК
- Получение списка подключённых устройств
- Статус подключения

---

## Требования

- Операционная система: Windows 10/11 (x86_64 / x86)
- 1С:Предприятие 8.3 (любая версия, начиная с 8.3.6)
- Установленные драйверы USB для Android-устройства
- Включённый режим отладки USB на Android-устройстве
- Android 5.0 (API 21) и выше

## Сборка

### Через CMake (CLI)

```bash
mkdir build && cd build
cmake -G "Visual Studio 16" ..
cmake --build . --config Release
```

### Через Visual Studio

1. Откройте `innNative.sln` или запустите CMake из Visual Studio
2. Выберите конфигурацию `Release|x64` или `Release|Win32`
3. Соберите проект

### Выходные файлы

После сборки DLL помещается в папку `1C\DEBUG\` или `1C\Release\`:

| Конфигурация | Платформа | Файл |
|--------------|-----------|------|
| Debug | Win32 | `1C\DEBUG\innNative_32.dll` |
| Debug | x64 | `1C\DEBUG\innNative_64.dll` |
| Release | Win32 | `1C\Release\innNative_32.dll` |
| Release | x64 | `1C\Release\innNative_64.dll` |

## Использование из 1С:Предприятие 8.3

### Подключение компоненты

```bsl
Драйвер = Новый ЗагрузитьВнешнююКомпоненту(
    "C:\path\to\ADBFileDriver.dll",
    "ADBFileDriver",
    ТипВнешнейКомпоненты.NativeAPI
);
```

### Подключение устройства

```bsl
Если Драйвер.ПодключитьУстройство() Тогда
    Сообщить("Устройство подключено: " + Драйвер.СтатусПодключения);
Иначе
    Сообщить("Ошибка подключения: " + Драйвер.СтатусПодключения);
КонецЕсли;
```

### Выгрузка файла на телефон

```bsl
Результат = Драйвер.ЗаписатьНаТелефон("C:\data\report.csv", "/Download/report.csv");
Если Результат Тогда
    Сообщить("Файл выгружен успешно");
КонецЕсли;
```

### Скачивание файла с телефона

```bsl
Содержимое = Драйвер.ПрочитатьСТелефона("/Download/report.csv");
Если Не Неопределено(Содержимое) Тогда
    Сообщить("Содержимое: " + Содержимое);
КонецЕсли;
```

### Получение списка устройств

```bsl
СписокJSON = Драйвер.ПолучитьСписокУстройств();
Сообщить(СписокJSON);
```

## Свойства

| Свойство | Тип | Описание |
|----------|-----|----------|
| `СтатусПодключения` | Текст | Текущий статус подключения |
| `СписокУстройств` | Текст (JSON) | Список устройств в JSON-формате |

---

## Методы

| Метод | Параметры | Возврат | Описание |
|-------|-----------|---------|----------|
| `ПодключитьУстройство()` | — | `bool` | Подключение к первому доступному устройству |
| `ОтключитьУстройство()` | — | `bool` | Отключение от устройства |
| `ЗаписатьНаТелефон(ПутьНаПК, ПутьНаТелефоне)` | `text, text` | `bool` | adb push файла на устройство |
| `ПрочитатьСТелефона(ПутьНаТелефоне)` | `text` | `text` | adb pull файла с чтением содержимого |
| `ПолучитьСписокУстройств()` | — | `text` | Список устройств в JSON-формате |
